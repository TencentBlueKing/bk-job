# Rules 规则库

本目录存放可被多个 Agent 复用的 Rules 规则定义。

## 什么是 Rules？

Rules 是一组行为规范或处理流程的定义，用于：
- 标准化 Agent 的输出格式
- 定义特定场景下的处理流程
- 约束 Agent 的行为边界

|---------|-------|------|
| 任务耗时分布阶段 | `任务耗时分布几个阶段.md` | 定义作业平台任务执行的六个阶段模型，用于耗时分析时的阶段划分 |
| 回答带日志平台链接 | `回答用户时带上日志平台查询链接.md` | 要求 Agent 在回答时必须附带日志平台的查询链接，方便用户自行查看 |

| 日志查询语句编写指导 | `日志查询语句编写指导.md` | KQL 语法指导，包含常用的日志查询语句模板和编写规范 |

## 使用方式

在 Agent 的 Prompt 中引用 Rule：
1. 直接将 Rule 内容嵌入 Prompt
2. 或在 Prompt 中引用 Rule 文件路径

## 添加新 Rule

1. 在本目录下创建新的 `.md` 文件
2. 按照现有格式编写规则内容
3. 更新本 README 中的规则列表
